Get Started

Don't waste your time.

With happyunicorn.css you can edit and manipulate your html styles directly in the css file!

No more adding a bunch of classes and memorizing padding and margin numbers

The hu.css is made using SASS, one of the most used css libraries on the web. All the html content is manipulated trought the scss file and cimpliled in CSS

You just need the create the classes in the HTML file and make all modification in the css, or no classes at all!

Caracteristics

  • No need to memorize hundreds of classes
  • Simples application
  • @include based: You import the styles using @iclude insted of classes

What you need to start

In order to user our library you need onlye some knowleged about CSS and HTML. But you can handle well just with the basics and Sass is easy to learn. Check the Sass oficial site to get start.
